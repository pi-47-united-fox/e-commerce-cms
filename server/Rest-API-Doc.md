&nbsp;

## HTTP Status Code
---
```
    200 - OK	Everything worked as expected.
    201 - Created	Everything worked as expected.
    403 - Forbidden	The access_token key doesn't have permissions to perform the request.
    --- 
    401 - Unauthorized	No valid access_token key provided.
    400 - Bad Request	The request was unacceptable, often due to missing a required parameter.
    404 - Not Found	The requested resource doesn't exist.
    500 - Server Errors	Something went wrong on Stripe's end. (These are rare.)
```

## EndPoints
---

#### User (Role Admin) Endpoints:
  1. POST /login

#### Product Endpoints:
  1. GET /products
  2. GET /products/:id
  3. POST /products/
  4. PUT /products/:id
  5. DELETE /products/:id

#### Banners Endpoints:
  1. GET /banners
  2. POST /banners/
  3. PUT /banners/:id
  4. DELETE /banners/:id

&nbsp;

## User Endpoints
---
### 1. POST /login
Request:
- data:
    ```json
    {
    "email": "string",
    "password": "string"
    }
    ```

Response:
- status: 200
- body:
    ```json
    {
        "access_token": "jwt string",
    }
    ```
